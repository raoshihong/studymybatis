mybatis插件：会拦截mybatis执行指定阶段的sql,并进行分析
只需实现 Interceptor 接口，并指定想要拦截的方法签名即可。
默认情况下，MyBatis 允许使用插件来拦截的方法调用包括：

Executor 类中的方法： (update, query, flushStatements, commit, rollback, getTransaction, close, isClosed)
ParameterHandler 类中的方法： (getParameterObject, setParameters)
ResultSetHandler 类中的方法： (handleResultSets, handleOutputParameters)
StatementHandler 类中的方法： (prepare, parameterize, batch, update, query)


mybatis插件的开发：
1.创建一个类实现Interceptor接口,并实现它的三个方法
2.使用@Intercepts标注拦截器接口
3.在@Intercepts使用方法签名@Signature(类,方法名,参数)
4.在mybatis的Configuration配置中添加拦截器
